About NutriScan

NutriScan is an Augmented Reality (AR) application designed to scan cards and display fruit objects interactively. It helps users learn nutritional information in a more engaging way.

Features

Interactive Scanning

Scan cards and view interactive 3D fruit models.

Nutrition Info

Learn about the benefits and nutrition of various fruits.

Mobile Friendly
